# Mohammad | Software Engineer & AI Specialist Portfolio

A premium, interactive, and responsive developer portfolio showcasing technical expertise in Java, Python, Spring Boot, FastAPI, Machine Learning, Cloud Development, and AI. Fully optimized with dark mode by default, dynamic API integrations, and visual assets.

---

## 🌟 Key Features

* **Developer Console Dashboard**:
  * **Dynamic GitHub Heatmap**: Renders your real contributions dynamically from a CORS-enabled scraper API, complete with custom tooltips, month/weekday labels, and green color scales.
  * **Dynamic LeetCode Heatmap**: Renders your submissions dynamically from your LeetCode profile calendar, styled matching GitHub but using orange color scales.
  * **Dynamic Streaks Calculations**: A JavaScript parsing engine automatically calculates and displays your **Current Streak**, **Longest Streak**, **Total Contributions**, **Global Ranking**, and **Active Days** in real-time.
  * **Fallback Simulator**: Generates realistic mockup data locally if the third-party APIs are rate-limited or offline, guaranteeing the UI never displays blank grids.
* **Auto-Rolling Infinite Marquees**:
  * Alternating marquee directions (right-to-left for Track A & C, left-to-right for Track B) preventing visual monotony.
  * Google & Professional Digital Badges showcase relocated as a section divider between the profile card and Developer Console dashboard.
* **Symmetric Interactive Project Showcase**:
  * Beautiful, balanced 2-column grid representing your signature projects:
    1. **Ishaara**: Indian Sign Language real-time recognition.
    2. **SHL Recommender**: Conversational recommendation system.
    3. **Apply Tracker**: Full-stack job search tracker dashboard.
    4. **MediScan**: Deep learning skin disease diagnosis panel.
  * Integration with modal lightboxes to inspect high-resolution project screens.
* **Timelines for Education & Experience**:
  * Flowing vertical timelines with connecting linear gradient lines, avoiding cluttering cards.
* **Premium UX Details**:
  * Default dark mode with an option to toggle to light mode.
  * Scroll progress bar indicators.
  * Glassmorphism effects with glowing cursor hover gradients.

---

## 🛠️ Technology Stack

* **Frontend Structure**: HTML5 (Semantic elements)
* **Styling**: Vanilla CSS3 (Custom variables, glassmorphism, keyframe animations)
* **Interactions**: Vanilla JavaScript (ES6+, Fetch API, asynchronous DOM rendering)
* **Icons & Logos**: FontAwesome 6, Devicons v2
* **Data Sources**:
  * GitHub: Scraped via `https://github-contributions-api.jogruber.de`
  * LeetCode: Fetched via `https://alfa-leetcode-api.onrender.com`

## 🚀 Deployment to Vercel

This project is a static site (HTML, CSS, and JavaScript) and can be deployed to Vercel with zero configuration:

1. Import the repository into your Vercel Dashboard.
2. Select **Framework Preset**: *Other* (or Vercel will automatically detect it as a static project).
3. Click **Deploy**.
